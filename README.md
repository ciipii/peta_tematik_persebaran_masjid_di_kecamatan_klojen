# Peta Tematik Persebaran Masjid di Kecamatan Klojen, Kota Malang

Proyek ini adalah sistem pemetaan berbasis web (**WebGIS**) yang menyajikan informasi lokasi dan persebaran masjid di wilayah Kecamatan Klojen. Proyek ini dikembangkan untuk memudahkan visualisasi sarana ibadah menggunakan teknologi pemetaan modern.
